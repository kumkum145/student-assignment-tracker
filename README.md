\# Student Assignment Tracker



A simple web application that helps students keep track of assignments.



\## Features

\- Add assignments

\- Display assignments in a list

\- Delete assignments



\## Technologies Used

\- HTML

\- CSS

\- JavaScript



\## How to Run

1\. Download the project files

2\. Open `index.html` in a browser
